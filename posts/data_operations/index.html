1 原地操作#

执行 X = X + Y 时，Python首先计算 Y + X ，为结果分配新的内存，然后使 Y 指向内存中的这个新位置。

print('before:id(X):', id(X))
X = X + Y
print('after:id(X):', id(X))

以上测试可以发现两次id(X)不一致。

如果要实现原地操作可以使用 X[:] = X + Y 或者 X += Y 。

2 张量对象转换#

torch定义的张量和numpy定义的张量可以很方便的相互转换。
torch.tensor -> numpy.ndarray

A = numpy.array([[1,2],[3,4]])
B = torch.tensor(A)             # 会拷贝数据，不共享内存
C = torch.from_numpy(A)         # 共享内存，原地修改一个会影响另一个，注意是原地修改！！
D = torch.as_tensor(A)          # 兼容各种输入（ndarray / list / Tensor）

numpy.ndarray -> torch.tensor

# tensor在CPU上，直接转换，共享内存
A = torch.tensor([[1, 2], [3, 4]])
B = A.numpy()
# tensor在GPU上，不能直接转换，需要先拷贝到 CPU 上再转换
A = torch.tensor([[1, 2], [3, 4]], device='cuda')
B = A.cpu().numpy()
# tensor 如果开启自动求导追踪 requires_grad=True，要先.detach()避免和计算图纠缠
A = torch.tensor([[1, 2], [3, 4]], requires_grad = True)
B = A.detach().numpy()

3 数据预处理#

数据处理一般是用`pandas，因为他可以与张量兼容。

3.1 读取数据#

第一步是从csv中获取到数据，但是这个数据一般还需要经过清洗等处理才能用来训练模型。
可以通过data = pd.read_csv(data_file)读取数据。
因为我们一般是用来做预测模型的训练，所以读取到数据之后一般还得将其分为输入和输出。
想要获取某一行/列或某几行/列的数据，可以用位置索引iloc。

例如：

inputs, outputs = data.iloc[:, 0:2], data.iloc[:, 2]

inputs为data的前两列，outputs为data的最后一列。

3.2 处理缺失值#

有些数据会显示NaN(not a number)，可能是在前期获取数据的时候获取不全面，这种情况是很常见的。 对于这些缺失值的处理，可以选择直接剔除缺失值或者对其进行插值等方法，主要取决于数据的特性。

我们有以下数据

NumRooms Alley
0 NaN Pave
1 2.0 NaN
2 4.0 NaN
3 NaN NaN

例如我们要用某一列其余数据的平均值填充空值

inputs = inputs.fillna(inputs.mean(numeric_only=True))      #numeric_only只对数值类型的列计算平均值

而对于inputs中的类别值或离散值，我们将“NaN”视为一个类别。

inputs = pd.get_dummies(inputs, dummy_na=True)

get_dummies() 可以吧分类变量转换成one-hot encoding
dummy_na 是表示是否把NaN当成一类
