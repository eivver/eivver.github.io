多层感知机

多层感知机在输出层和输入层之间增加一个或多个全连接隐藏层，并通过激活函数转换隐藏层的输出。

1 模型结构#

线性回归只有一层，且只能表示线性关系，这样的模型表达能力很有限。
很自然地，我们会想能不能增加网络的层数，将多个连接层堆叠，中间的层就叫隐藏层。 这样就能增强模型的表达能力了吗？
并不是，因为每一层都是线性的，最终也还是线性的，所以多层网络最终也可以变换成一层。
对于任意权重值， 我们只需合并隐藏层，便可产生具有参数 $\mathbf{W} = \mathbf{W}^{(1)}\mathbf{W}^{(2)}$ 和$\mathbf{b} = \mathbf{b}^{(1)} \mathbf{W}^{(2)} + \mathbf{b}^{(2)}$ 的等价单层模型： $$ \mathbf{O} = (\mathbf{X} \mathbf{W}^{(1)} + \mathbf{b}^{(1)})\mathbf{W}^{(2)} + \mathbf{b}^{(2)} = \mathbf{X} \mathbf{W}^{(1)}\mathbf{W}^{(2)} + \mathbf{b}^{(1)} \mathbf{W}^{(2)} + \mathbf{b}^{(2)} = \mathbf{X} \mathbf{W} + \mathbf{b}. $$ 因此我们引入激活函数，对每一层的输出做一个非线性变换，这样就可以通过层的堆叠产生更有表达能力的模型了。

1.2 激活函数#

激活函数（activation function） 通过计算加权和并加上偏置来确定神经元是否应该被激活，它们将输入信号转换为输出的可微运算。大多数激活函数都是非线性的。

以下是几种常用激活函数:
ReLU(Rectified linear unit) $$\operatorname{ReLU}(x) = \max(x, 0).$$

  • 没有指数，计算简单

sigmoid $$\operatorname{sigmoid}(x) = \frac{1}{1 + \exp(-x)}.$$

tanh $$\operatorname{tanh}(x) = \frac{1 - \exp(-2x)}{1 + \exp(-2x)}.$$

2 模型实现#

# 模型结构
net = nn.Sequential(
    nn.Flatten(),
    nn.Linear(28*28,256),
    nn.ReLU(),              # 激活层
    nn.Linear(256,10))

# 初始化参数
def init_weights(m):
    if type(m) == nn.Linear:
        nn.init.normal_(m.weight, std = 0.01)
net.apply(init_weights)

batch_size, lr, num_epochs = 256, 0.1, 10
loss = nn.CrossEntropyLoss(reduction = 'none')
trainer = torch.optim.SGD(net.parameters(), lr = lr)

train_iter, test_iter = d2l.load_data_fashion_mnist(batch_size)
# 用的d2l库里的
d2l.train_ch3(net, train_iter, test_iter, loss, num_epochs, trainer)

3 过拟合#

将模型在训练数据上拟合的比在潜在分布中更接近的现象称为过拟合（overfitting），用于对抗过拟合的技术称为正则化（regularization）。
权重衰退（也叫$L_2$正则化）和Dropout是常用的两种正则技术。

3.1 权重衰退#

在线性回归中，我们的损失是： $$L(\mathbf{w}, b) = \frac{1}{n}\sum_{i=1}^n \frac{1}{2}\left(\mathbf{w}^\top \mathbf{x}^{(i)} + b - y^{(i)}\right)^2$$ 给他加入一个$| \mathbf{w} |^2$来度量模型的复杂性，类似于在最小化损失的同时，也尽量使模型的参数不那么复杂，相当于加入了一个惩罚项。然后通过一个常数来调整二者的权衡。
$$L(\mathbf{w}, b) + \frac{\lambda}{2} |\mathbf{w}|^2$$

$L_2$正则化回归的小批量随机梯度下降更新如下式： $$ \begin{aligned} \mathbf{w} & \leftarrow \left(1- \eta\lambda \right) \mathbf{w} - \frac{\eta}{|\mathcal{B}|} \sum_{i \in \mathcal{B}} \mathbf{x}^{(i)} \left(\mathbf{w}^\top \mathbf{x}^{(i)} + b - y^{(i)}\right). \end{aligned} $$

$\eta$和$\lambda$一般都是小于1的，因此$(1- \eta\lambda)<1$也就是每次更新都在尝试缩小权重$\mathbf{w}$，这就是权重衰退。

因此我们在实现的时候只需要改变优化器的优化方法，而没必要去改变损失函数。

trainer = torch.optim.SGD([
    {"params":net[0].weight,'weight_decay': weight_decay},
    {"params":net[0].bias}], lr=lr)

我们在实例化优化器时直接通过weight_decay指定weight decay超参数。
默认情况下，PyTorch同时衰减权重和偏移。
这里我们只为权重设置了weight_decay，所以偏置参数不会衰减。

3.2 Dropout#


就是在训练的过程中把某些神经元丢弃，其实就是随机注入噪声，现在的主流观点认为是一种正则化。

dropout1, dropout2 = 0.2, 0.5
net = nn.Sequential(nn.Flatten(),
        nn.Linear(784, 256),
        nn.ReLU(),
        # 在第一个全连接层之后添加一个dropout层
        nn.Dropout(dropout1),
        nn.Linear(256, 256),
        nn.ReLU(),
        # 在第二个全连接层之后添加一个dropout层
        nn.Dropout(dropout2),
        nn.Linear(256, 10))

def init_weights(m):
    if type(m) == nn.Linear:
        nn.init.normal_(m.weight, std=0.01)

net.apply(init_weights)

直接在全连接层后加一个Dropout层就好了

值得注意的是，Dropout层只在训练期间有效，在进行推理的时候相当于不存在。
