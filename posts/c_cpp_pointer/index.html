顺时针螺旋移动法#

  1. 找到第一个未定义的标识符，即变量名，以他为中心
  2. 顺时针螺旋移动
    • [] or [N] 一个大小为N的数组，元素类型是
    • [P] [Q] P行Q列的二维数组，元素类型是…
    • () 一个函数，形参是()，返回值类型是..
    • * 一个指针，指向
    • & 一个引用，指向
    • const 常量的，（const T和T const是等价的）
  3. 重复2先解析完括号里的再跳出括号

嵌套声明举例#

  • int (*bar)[5]：bar是一个指针，指向一个大小为5的数组，元素类型是int
  • char const *p(int ,int * const *)：p是一个函数，函数的形参是int和（一个指针指向一个常量的指针，这个指针指向一个int），返回值是一个指针，这个指针指向常量的char。
  • void (*signal(int, void (*)(int)))(int)：signal是一个函数，函数的形参是int和（一个指针指向一个函数，形参是int，返回值是void），函数返回一个指针，指向一个函数，函数形参是int，返回值是void。
  • char **argv：argv是一个指针，这个指针指向（一个指针，这个指针指向一个char类型的变量），一般用来实现C语言主函数的参数传递，这是一个二级指针，在main函数中，它指向一个指针数组的首元素地址。该数组的每个元素都是一个 char * 指针，分别指向对应的命令行参数字符串（以 \0 结尾）。
  • char *argv[]：这个和上面的char **argv是等价的，因为C语言函数传递参数的时候数组会退化成指针，这样表示可以更好理解第一级为什么是指向一个数组的指针。

const的几种情形#

const 修饰它左边的东西，如果左边没有，就修饰右边

  • int const &A：A是一个引用，指向常量的int
  • int* const B：B是一个常量的指针，指向int
  • int const *C：C是一个指针，指向常量的int
  • int const* const D：D是一个常量的指针，指向常量的int

使用 typedef 简化#

void (*signal(int, void (*)(int)))(int) 太复杂了，一般可以用typedef简化

typedef void (*Handler)(int);
Handler signal(int sig, Handler func);
