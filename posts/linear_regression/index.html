线性回归

线性回归是一种最基础的监督学习方法，用来研究输入特征和输出变量之间的线性关系。

回归（regression）是能为一个或多个自变量与因变量之间关系建模的一类方法。
线性回归（linear regression）基于几个简单的假设：
首先，假设自变量$\mathbf{x}$和因变量$y$之间的关系是线性的， 即$y$可以表示为$\mathbf{x}$中元素的加权和，这里通常允许包含观测值的一些噪声；
其次，我们假设任何噪声都比较正常，如噪声遵循正态分布。

1 线性模型#

1.1 多变量线性模型#

对于单变量的线性模型我们在高中就学过了，实际问题一般会对应多个自变量(比如房屋价格和面积，房龄，地段等特征有关)。
而要把单变量拓展到多变量也很简单，我们通常使用线性代数来表示，也会方便后面的计算。当我们的输入包含$d$个特征时，我们将预测结果$\hat{y}$ （通常使用“尖角”符号表示$y$的估计值）表示为：

$$\hat{y} = w_1 x_1 + … + w_d x_d + b$$

将所有特征放到向量$\mathbf{x} \in \mathbb{R}^d$中， 并将所有权重放到向量$\mathbf{w} \in \mathbb{R}^d$中， 我们可以用点积形式来简洁地表达模型：

$$\hat{y} = \mathbf{w}^\top \mathbf{x} + b$$

向量$\mathbf{x}$对应于单个数据样本的特征。

进一步的，考虑数据集中的所有样本，用符号表示的矩阵$\mathbf{X} \in \mathbb{R}^{n \times d}$ 可以很方便地引用我们整个数据集的$n$个样本。 其中，$\mathbf{X}$的每一行是一个样本，每一列是一种特征。
对于特征集合$\mathbf{X}$，预测值$\hat{\mathbf{y}} \in \mathbb{R}^n$ 可以通过矩阵-向量乘法表示为：

$${\hat{\mathbf{y}}} = \mathbf{X} \mathbf{w} + b$$

  • $\mathbf{w}$为权重，$b$为偏置
  • 我们要找到合适的$\mathbf{w}$和$b$使得预测的$\hat{\mathbf{y}}$与实际的$\mathbf{y}$最接近

1.2 损失函数#

要找到合适的$\mathbf{w}$和$b$使得预测的$\hat{\mathbf{y}}$与实际的$\mathbf{y}$最接近，我们就需要确定一个度量来判断二者的接近程度。损失函数（loss function）能够量化目标的实际值与预测值之间的差距。
通常我们会选择非负数作为损失，且数值越小表示损失越小，完美预测时的损失为0。
回归问题中最常用的损失函数是平方误差函数，可以通过极大似然估计来推导。 当样本$i$的预测值为$\hat{y}^{(i)}$，其相应的真实标签为$y^{(i)}$时， 平方误差可以定义为以下公式：

$$l^{(i)}(\mathbf{w}, b) = \frac{1}{2} \left(\hat{y}^{(i)} - y^{(i)}\right)^2.$$

考虑所有样本的损失和，写成矩阵形式：

$$L(\mathbf{w}, b) =\frac{1}{n}\sum_{i=1}^n l^{(i)}(\mathbf{w}, b) =\frac{1}{n} \sum_{i=1}^n \frac{1}{2}\left(\mathbf{w}^\top \mathbf{x}^{(i)} + b - y^{(i)}\right)^2.$$

1.3 随机梯度下降#

假设我们得不到解析解，那我们要怎么去“猜”我们需要的权重和偏置呢？
梯度下降（gradient descent）几乎可以优化所有深度学习模型。它通过不断地在损失函数递减的方向上更新参数来降低误差。

梯度下降最简单的用法是计算损失函数（数据集中所有样本的损失均值）关于模型参数的导数（在这里也可以称为梯度）。但实际中的执行可能会非常慢：因为在每一次更新参数之前，我们必须遍历整个数据集。因此，我们通常会在每次需要计算更新的时候随机抽取一小批样本，这种变体叫做小批量随机梯度下降（minibatch stochastic gradient descent）。

在每次迭代中，我们首先随机抽样一个小批量$\mathcal{B}$，它是由固定数量的训练样本组成的。然后，我们计算小批量的平均损失关于模型参数的导数（也可以称为梯度）。最后，我们将梯度乘以一个预先确定的正数$\eta$，并从当前参数的值中减掉。

我们用下面的数学公式来表示这一更新过程（$\partial$表示偏导数）：

$$(\mathbf{w},b) \leftarrow (\mathbf{w},b) - \frac{\eta}{|\mathcal{B}|} \sum_{i \in \mathcal{B}} \partial_{(\mathbf{w},b)} l^{(i)}(\mathbf{w},b).$$

算法的步骤如下：

  • （1）初始化模型参数的值，如随机初始化；
  • （2）从数据集中随机抽取小批量样本且在负梯度的方向上更新参数，并不断迭代这一步骤。

1.4 解析解#

实际上线性回归是我们少数几个可以得到确定解析解的模型，以下是推导过程。

因为$\mathbf{w}$和$b$都是待求的参数，我们可以把$w\mathbf{w}$和$b$吸收入向量形式$\tilde{\mathbf{w}} = (\mathbf{w};b)$ ，相应的，样本矩阵也要变成$\tilde{\mathbf{X}}$，每行的最后一个元素恒为1，即： $$ \tilde{\mathbf{X}} = \begin{bmatrix} \mathbf{x}_1^\top & 1 \\ \mathbf{x}_2^\top & 1 \\ \vdots & \vdots \\ \mathbf{x}_n^\top & 1 \end{bmatrix} \in \mathbb{R}^{n \times (d+1)} $$

于是预测值就可以简洁地写成：

$$ \hat{\mathbf{y}} = \tilde{\mathbf{X}} \tilde{\mathbf{w}} $$

使均方误差最小化可以得到：

$$ \tilde{\mathbf{w}}^\ast = (\tilde{\mathbf{X}}^\top \tilde{\mathbf{X}})^{-1} \tilde{\mathbf{X}}^\top \mathbf{y} $$

像线性回归这样的简单问题存在解析解，但并不是所有的问题都存在解析解。 解析解可以进行很好的数学分析，但解析解对问题的限制很严格，导致它无法广泛应用在深度学习里。

1.5 神经网络图#

这样的线性模型在深度学习中可以视为一个单层神经网络。

在这个神经网络中，输入为$x_1, \ldots, x_d$，因此输入层中的输入数（或称为特征维度，feature dimensionality）为$d$。网络的输出为$o_1$，因此输出层中的输出数是1。
需要注意的是，输入值都是已经给定的，并且只有一个计算神经元。 由于模型重点在发生计算的地方，所以通常我们在计算层数时不考虑输入层。 也就是说，该神经网络的层数为1。我们可以将线性回归模型视为仅由单个人工神经元组成的神经网络，或称为单层神经网络。

对于线性回归，每个输入都与每个输出（在本例中只有一个输出）相连， 我们将这种变换称为全连接层（fully-connected layer）或称为稠密层（dense layer）。

2 模型的实现#

我们可以使用PyTorch的高级API简洁地实现模型。
在PyTorch中，data模块提供了数据处理工具，nn模块定义了大量的神经网络层和常见损失函数。

2.1 生成数据#

为了简单起见，我们将根据带有噪声的线性模型构造一个人造数据集。

def synthetic_data(w, b, num_examples):  #@save
    """生成y=Xw+b+噪声"""
    X = torch.normal(0, 1, (num_examples, len(w)))
    y = torch.matmul(X, w) + b
    y += torch.normal(0, 0.01, y.shape)
    return X, y.reshape((-1, 1))        # 生成的y原本是一维向量(1000)，将其reshape成(1000,1)

true_w = torch.tensor([-1, 3.4])
true_b = 4.2
features, labels = synthetic_data(true_w, true_b, 1000)

# 画出来看看
d2l.set_figsize()
d2l.plt.scatter(features[:, (0)].detach().numpy(), labels.detach().numpy(), 1)
d2l.plt.scatter(features[:, (1)].detach().numpy(), labels.detach().numpy(), 1)

2.2 读取数据#

我们可以调用框架中现有的API来读取数据。 我们将features和labels作为API的参数传递，并通过数据迭代器指定batch_size。

from torch.utils import data
def load_array(data_arrays, batch_size, is_train=True):  #@save
    """构造一个PyTorch数据迭代器"""
    dataset = data.TensorDataset(*data_arrays)
    return data.DataLoader(dataset, batch_size, shuffle=is_train)

batch_size = 10
data_iter = load_array((features, labels), batch_size)

代码解释：

  1. TensorDataset是PyTorch提供的一个数据集类，它可以把多个张量按样本对应起来。
    可以这么简单的理解：dataset[0] == (feature[0]，labels[0])， dataset[1] == (feature[1]，labels[1]) …
    也就是说，TensorDataset把特征和标签一一对应绑定，形成一个“数据集”。
  2. *data_arrays的*表示拆包，这里就等价于我们传进去的(features, labels)。
  3. DataLoader是PyTorch提供的一个迭代器，能从数据集里分批次（mini-batch）取数据。
    其中shuffle=True：是否随机打乱数据,
    得到的data_iter是一个可迭代对象。可以这样使用： 
    for X, y in data_iter:
    print(X, y)   # X 是一批特征，y 是对应的标签
    • 每次返回的数量是batch_size
    • 如果我们的data_iter的shuffle属性为True，每次返回的数据是随机的，否则为固定的

2.3 定义模型#

对于标准深度学习模型，我们可以使用框架的预定义好的层。这使我们只需关注使用哪些层来构造模型，而不必关注层的实现细节。

# nn是神经网络(Neural Network)的缩写
from torch import nn
net = nn.Sequential(nn.Linear(2, 1))

nn.Linear 是全连接层也叫线性层，实现了一个最基本的仿射变换： $$y = xW^\top + b$$ 输入参数：

  • in_features：输入特征维度
  • out_features：输出特征维度
  • bias（可选，默认 True）：是否加偏置 $b$

eg: nn.Linear(2, 1)表示输入2维输出1维，默认带偏置

nn.Sequential 是层的容器，可以把多个层按顺序“拼接”起来，当给定输入数据时，Sequential实例将数据传入到第一层， 然后将第一层的输出作为第二层的输入，以此类推。 eg：

model = nn.Sequential(
    nn.Linear(2, 3),
    nn.ReLU(),          #激活函数，对数据做了个非线性变换
    nn.Linear(3, 1)
)

以上的模型就会对输入的x进行： x → Linear(2→3) → ReLU → Linear(3→1) → 输出

我们的模型只包含一个层，因此实际上不需要Sequential。但是由于以后几乎所有的模型都是多层的，在这里使用Sequential提前熟悉“标准的流水线”。

2.4 初始化参数#

在对我们的网络进行训练之前还需要对待训练的权重weight和bias先进行初始化，如果所有权重初始化为 0，那么每个神经元的梯度完全一样，参数更新过程没有区别，网络就退化了，相当于只有一个神经元在学习。
一般在调用nn.Linear(2, 1)的时候会有默认方案初始化，但是为了养成好习惯，还是需要手动初始化一下。
以下是几种常见的初始化方法：

方法 分布 适用激活 特点
Normal 正态 一般 简单、可控 std
Uniform 均匀 一般 简单、可控范围
Xavier / Glorot 正态或均匀 Sigmoid/Tanh 保持方差稳定
He / Kaiming 正态或均匀 ReLU 考虑 ReLU 异质性，防梯度消失
Orthogonal 正交矩阵 RNN 保持信息传递稳定

我们这里就直接使用正态分布初始化了。

net[0].weight.data.normal_(0, 0.01)
net[0].bias.data.fill_(0)

我们通过net[0]选择网络中的第一个图层，
然后使用weight.data和bias.data方法访问参数。
我们还可以使用替换方法normal_和fill_来重写参数值。

2.5 损失函数#

常用的损失函数有这些：

任务类型 常用损失函数 PyTorch 类
回归 MSE, MAE, Huber MSELoss, L1Loss, SmoothL1Loss
二分类 BCE BCELoss, BCEWithLogitsLoss
多分类 Cross Entropy CrossEntropyLoss
序列/语言 NLL, CrossEntropy NLLLoss, CrossEntropyLoss
度量学习 Cosine, Contrastive, Triplet CosineEmbeddingLoss, 自定义
生成模型 BCE, Wasserstein BCELoss, 自定义

正如我们在前面所讨论的，回归问题一般使用均方误差作为损失函数。
计算均方误差使用的是MSELoss类，也称为平方$L_2$范数。
默认情况下，它返回所有样本损失的平均值。

loss = nn.MSELoss()

2.6 优化算法#

常用的优化算法：

优化器 核心思想 适用场景
SGD 每次迭代用梯度下降更新参数 小模型、基础学习
SGD + Momentum 利用历史梯度加速收敛，减少震荡 CNN、卷积网络
Nesterov 预先修正梯度方向，提前“感知”未来更新 CNN、深层网络
Adagrad 每个参数独立学习率，对稀疏梯度自适应调整 稀疏特征、NLP
RMSProp 对梯度平方做指数加权平均，自适应调整学习率 RNN、LSTM
Adam 结合动量和自适应学习率，快速收敛 大多数深度学习任务
AdamW Adam + 权重衰减正则化，提高泛化能力 Transformer、NLP
Nadam Adam + Nesterov 动量 特殊深层网络

小批量随机梯度下降算法是一种优化神经网络的标准工具，PyTorch在optim模块中实现了该算法的许多变种。
当我们实例化一个SGD实例时，我们要指定优化的参数（可通过net.parameters()从我们的模型中获得）以及优化算法所需的超参数字典。
小批量随机梯度下降只需要设置lr值，这里设置为0.03。

trainer = torch.optim.SGD(net.parameters(), lr=0.03)

2.7 训练模型#

当我们完成前面的准备工作，得到了数据集，模型，损失函数，优化器就可以开始对我们的模型进行训练了。

num_epochs = 3
for epoch in range(num_epochs):
    for X, y in data_iter:
        l = loss(net(X) ,y)         # net(X)也即预测的y
        trainer.zero_grad()         # 梯度清零
        l.backward()                # 反向传播
        trainer.step()              # 更新参数
    l = loss(net(features), labels)
    print(f'epoch {epoch + 1}, loss {l:f}')

trainer = torch.optim.SGD(net.parameters(), lr=0.03) 是在 PyTorch 里创建一个 优化器对象，它负责更新参数。torch.optim.SGD 继承自 torch.optim.Optimizer，因此它内部的方法和属性基本上和所有优化器通用。

常用方法

  • step()：遍历参数组 → 遍历参数 → 按算法更新参数
  • zero_grad()：清空所有参数的梯度，避免梯度累积。
  • state_dict()/load_state_dict()：保存/恢复优化器的全部信息（学习率、动量缓存等）
  • add_param_group()：动态加一组参数进来，常用于微调时给新层加优化策略。
